🔑 GitHub 연결 설정

토큰 만드는 법

  1. github.com/settings/personal-access-tokens/new 접속 후 로그인
  2. Repository access → "Only select repositories" → 이 저장소 선택
  3. Permissions → Repository permissions → Contents: Read and write 로 설정
  4. Generate token → 생성된 토큰을 복사해서 위 칸에 붙여넣기

토큰은 이 기기의 브라우저(localStorage)에만 저장되고, GitHub API 외에는 어디로도 전송되지 않습니다.
